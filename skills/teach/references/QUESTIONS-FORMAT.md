# Questions Format

`QUESTIONS.md` tracks open investigations — questions the user is actively exploring. Unlike confusions (which are gaps to fill), these are lines of inquiry.

## Template

```md
# Questions I'm Investigating

- {Question} — emerged from [[Related Concept]]
- {Question} — emerged from [[Related Concept]]
```

## Example

```md
# Questions I'm Investigating

- Why are [[AI Agents]] structured as tools? — emerged from [[MCP Tool Calling]]
- Why does [[JSON-RPC]] keep appearing across different systems? — emerged from [[MCP]]
- What makes some abstractions durable while others collapse? — emerged from [[API Design]]
```

## Rules

- **Questions are discoveries.** Add one when a lesson raises something worth exploring later. This is different from confusion — it's active curiosity.
- **Link to origin.** Every question links to the concept that sparked it.
- **Review monthly.** During synthesis, check if any questions now have answers.
- **Keep even unanswered ones.** Many important insights emerge months later when several questions converge.
