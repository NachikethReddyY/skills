# Teach-Back Format

Teach-backs are the final exam. If the user can explain a concept at multiple depths without notes, they truly understand it. Teach-backs live in `{topic}/records/` as numbered records.

## Template

```md
# Teach-Back: {Concept}

## 30 Seconds
{The elevator pitch — what it is, why it matters.}

## 5 Minutes
{The overview — key concepts, how it works, when to use it. Enough to understand the basics.}

## 15 Minutes
{The deep dive — mechanics, tradeoffs, edge cases, history, connections. Include `[[wiki-links]]` to related concepts.}
```

## Rules

- **Every major concept gets a teach-back.** Schedule for Day 7 of reviews.
- **Don't write it for the user.** They produce it from memory. Ask clarifying questions but don't supply content.
- **Three depths are mandatory.** Each exercises different retrieval paths.
- **Grade it.** Was it accurate? Complete? Did it connect to other concepts?
- **Use `[[wiki-links]]`.** The teach-back is a knowledge artifact — link to related concepts.
